# BCL2-natural-compounds-docking
Computational screening of natural compounds against BCL-2 protein using molecular docking and bioinformatics analysis.

## Project Overview
This project performs molecular docking of natural compounds against the BCL-2 protein to identify potential inhibitors.

## Workflow
1. Data acquisition
2. Protein structure preparation
3. Molecular docking
4. Docking analysis and visualization

## Tools Used
- Python
- AutoDock Vina
- Pandas
- Matplotlib

## Compounds Tested
- Curcumin
- Quercetin
- Resveratrol
- Apigenin
- Berberine

## Output
- Docking scores
- Interaction analysis
- Drug-likeness properties
